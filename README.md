ilo
======

R tools to access data from [ILO Statistics department](http://www.ilo.org/ilostat/). The package offers tool to search, download and manipulate data from the ilostat database.

For installation and usage, check the [tutorial page](https://github.com/ilo/vignettes/Start.md).  

You can install:

-   the latest development version from github with

    ``` r
    if (packageVersion("devtools") < 1.6) {install.packages("devtools")}
    devtools::install_github("dbescond/ilo")
    ```
